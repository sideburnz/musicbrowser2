MusicBrowser 2 is a free WMC plug-in that allows you to browse your music collection by folders.
  • Download version 2.2.2
  • Get Support
  • Knowledge Base

Features

  • Automatic metadata retrieval
  • Create random / full folder playlists
  • Doesn't rely on your music being tagged
  • Works with a range of music formats

Designed for Windows Media Center 7

MusicBrowser 2 is an open source software project

MusicBrowser 2 uses images from HT Backdrops

MusicBrowser 2 uses images and data from Last.fm

Source code also available on GoogleCode
